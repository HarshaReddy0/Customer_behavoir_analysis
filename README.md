# 📊 Customer Behavior Data Analytics Project

## Overview

This project demonstrates a complete end-to-end Data Analytics workflow using Python, PostgreSQL, and Power BI. The project focuses on transforming raw customer data into meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL analysis, dashboard development, and reporting.

The goal is to support data-driven decision-making by identifying customer behavior patterns, sales trends, and business opportunities.

---

## Dataset

The dataset contains customer and sales-related information, including:

* Customer demographics
* Purchase transactions
* Product details
* Sales and revenue data
* Customer behavior metrics

The data was analyzed to uncover trends, customer segments, and performance indicators.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Jupyter Notebook using Python.
* Examined dataset structure and column information.
* Verified data quality and completeness.

### 2. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent entries.
* Standardized data formats.

### 3. Exploratory Data Analysis (EDA)

Performed detailed analysis using Python in Jupyter Notebook:

* Descriptive statistics
* Data distribution analysis
* Correlation analysis
* Customer behavior analysis
* Sales trend analysis
* Data visualization using charts and graphs

### 4. SQL Analysis (PostgreSQL)

Imported the cleaned dataset into PostgreSQL and executed SQL queries to generate business insights.

Key analyses included:

* Total sales calculation
* Revenue analysis
* Customer segmentation
* Top-performing products
* Monthly sales trends
* Customer purchase behavior

### 5. Dashboard Development (Power BI)

Created an interactive Power BI dashboard to visualize key business metrics.

Dashboard features:

* Total Sales KPI
* Revenue Analysis
* Customer Insights
* Product Performance
* Sales Trend Analysis
* Interactive Filters and Slicers

## How to Run

### Python Analysis

1. Open Jupyter Notebook.
2. Load the dataset.
3. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

4. Run all notebook cells to perform data cleaning and EDA.

### PostgreSQL Analysis

1. Create a database in PostgreSQL.
2. Import the cleaned dataset.
3. Execute SQL queries from the SQL script file.
4. Analyze the generated results.

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the data source.
3. Explore dashboard visuals and insights.
